# OLED IVL & LT Analyzer 설명서

## 1. 개요

OLED 소자의 **IVL (전류-전압-휘도)** 및 **LT (수명)** 데이터를 CSV 파일로 업로드하면,
자동으로 분석하여 **비교 테이블**과 **차트**를 생성하는 웹 기반 분석 도구입니다.

- **버전**: v2.0
- **담당**: jhbaik@ltml.co.kr (LT소재 품질경영팀)
- **기술 스택**: 순수 HTML/CSS/JS (단일 파일), Chart.js v4

---

## 2. 전체 구조

```
┌──────────────────────────────────────────┐
│  HEADER (로고 + 타이틀)                    │
├──────────────────────────────────────────┤
│  FILE UPLOAD (드래그 앤 드롭 영역)          │
├──────────────────────────────────────────┤
│  FILE MATCHING (8개 슬롯 그리드, 2열×4행)   │
├──────────────────────────────────────────┤
│  ANALYZE 버튼                             │
├──────────────────────────────────────────┤
│  IVL ANALYSIS 테이블 (메인)                │
├──────────────────────────────────────────┤
│  IVL ANALYSIS 테이블 (추가 비교)            │
├──────────────────────────────────────────┤
│  LT CHART (수명 그래프)                    │
├──────────────────────────────────────────┤
│  IVL SPECTRUM 차트 (@J=10mA/cm²)          │
├──────────────────────────────────────────┤
│  FOOTER                                  │
└──────────────────────────────────────────┘
```

## 3. 파일 슬롯 구성 (8개)

| 슬롯 키        | 라벨           | 타입 | 그룹    | 번호 |
|---------------|---------------|------|--------|------|
| REF_IVL1      | REF IVL-1     | IVL  | REF    | 1    |
| REF_IVL2      | REF IVL-2     | IVL  | REF    | 2    |
| SAMPLE_IVL1   | SAMPLE IVL-1  | IVL  | SAMPLE | 1    |
| SAMPLE_IVL2   | SAMPLE IVL-2  | IVL  | SAMPLE | 2    |
| REF_LT1       | REF LT-1      | LT   | REF    | 1    |
| REF_LT2       | REF LT-2      | LT   | REF    | 2    |
| SAMPLE_LT1    | SAMPLE LT-1   | LT   | SAMPLE | 1    |
| SAMPLE_LT2    | SAMPLE LT-2   | LT   | SAMPLE | 2    |

- 최소 **REF IVL 1개 + SAMPLE IVL 1개** 또는 **LT 파일 1개** 이상이어야 분석 가능
- 4~8개 파일을 한번에 업로드 가능

## 4. 기능별 상세 설명

### 4.1 FILE UPLOAD (파일 업로드)

- **드래그 앤 드롭**: CSV 파일을 드롭존 영역에 끌어다 놓기
- **클릭 업로드**: 영역 클릭 시 파일 선택 다이얼로그 오픈
- `.csv` 확장자만 허용
- 인코딩 자동 감지: UTF-8 우선, 실패 시 EUC-KR로 재시도

### 4.2 FILE MATCHING (파일 자동 매칭)

파일명을 분석하여 8개 슬롯에 자동 배정합니다.

**매칭 규칙 (`detect` 함수)**:
1. 파일명 앞의 특수문자(★, * 등) 제거
2. `ref_` 또는 `sample_` (sam_ 포함)로 시작하는지 판별 → 그룹 결정
3. `ivl` 또는 `lt` 키워드와 숫자(1/2)로 타입 및 번호 결정
4. 예: `REF_IVL1_xxx.csv` → REF IVL-1 슬롯에 배정

**매칭 실패/중복 처리**:
- 매칭 실패 파일: 노란색 경고 박스에 목록 표시
- 중복 매칭: 빨간색 경고 박스, 마지막 파일로 덮어쓰기
- 수동 매칭: 각 슬롯의 📎 버튼으로 개별 파일 직접 선택 가능

**레이아웃**: 2열 × 4행 그리드 (긴 파일명 대응)

### 4.3 IVL ANALYSIS 테이블

CSV의 **J(전류밀도) ≈ 10 mA/cm²** 지점 데이터를 추출하여 비교합니다.

**추출 항목** (CSV 컬럼 기준):
| 항목 | CSV 인덱스 | 설명 |
|------|-----------|------|
| Op.V (V) | [1] | 동작 전압 |
| EL EFF (cd/A) | [7] | 전류 효율 |
| EQE (%) | [9] | 외부 양자 효율 |
| CIE(x) | [10] | 색좌표 x |
| CIE(y) | [11] | 색좌표 y |
| MAX 파장 | [13~] | 스펙트럼 최대 파장 |

**비교 블록 (메인 테이블)**:
| 블록 | REF | SAMPLE |
|------|-----|--------|
| ① REF1 vs SAMPLE1 | REF_IVL1 / REF_LT1 | SAMPLE_IVL1 / SAMPLE_LT1 |
| ② REF1 vs SAMPLE2 | REF_IVL1 / REF_LT1 | SAMPLE_IVL2 / SAMPLE_LT2 |
| ③ REF2 vs SAMPLE1 | REF_IVL2 / REF_LT2 | SAMPLE_IVL1 / SAMPLE_LT1 |
| ④ REF2 vs SAMPLE2 | REF_IVL2 / REF_LT2 | SAMPLE_IVL2 / SAMPLE_LT2 |

**비교 블록 (추가 테이블)** — 동일 그룹 내 비교:
| 블록 | 비교 대상 |
|------|----------|
| SAMPLE1 vs SAMPLE2 | SAMPLE 끼리 비교 |
| SAMPLE2 vs SAMPLE1 | 역방향 비교 |
| REF1 vs REF2 | REF 끼리 비교 |
| REF2 vs REF1 | 역방향 비교 |

**Result (비교 결과) 행**:
- 각 항목을 `SAMPLE / REF × 100%` 비율로 표시
- 전압만 `REF / SAMPLE × 100%` (낮을수록 좋으므로 반대)
- **±5% 이내**: 파란색 (정상), **±5% 초과**: 빨간색 (이탈)

### 4.4 LT (수명) 분석

**LT 레벨 판정 (`ltAll` 함수)**:
- LT99 ~ LT90까지 각 레벨별로 해당 휘도(%) 아래로 **영구 하락**한 시점(hr)을 탐색
- 뒤에서부터 역방향 스캔하여, 마지막으로 해당 레벨 이상이었던 지점의 다음 지점을 채택
- 측정 종료 시점까지 레벨 이상이면 `null` (아직 미도달)

**LT 모드 탭**:
| 모드 | 표시 레벨 | 추가 컬럼 |
|------|----------|----------|
| 95~90 (기본) | LT95, LT94, LT93, LT92, LT91, LT90 | 없음 |
| 99~96 | LT99, LT98, LT97, LT96 | 측정시간 |

**LT 비교 결과**: `SAMPLE LT / REF LT × 100%`
- 100h 미만 데이터는 `(100h 미만)` 경고 표시

**소수점 선택**: 정수 / 1자리(기본) / 2자리

### 4.5 LT CHART (수명 그래프)

Chart.js 기반 인터랙티브 라인 차트입니다.

**컨트롤 패널**:

| 컨트롤 | 설명 | 범위 |
|--------|------|------|
| X max | X축 최대값 (시간) | 10 ~ 데이터 최대×2 hr, 10 단위 |
| Y min | Y축 최소값 (%) | 90 ~ 100%, 0.5 단위 |
| Y max | Y축 최대값 (%) | 98 ~ 101%, 0.5 단위 |
| Smooth | 이동평균 스무딩 윈도우 | 1(OFF) ~ 15 |

**모드 버튼**:
- **WHITE**: 배경 흰색/검정 전환 (기본: 흰색)
- **SQUARE**: 정사각형 비율 전환 (기본: 정사각형, 최대 600px)

**축 버튼**:
- **X Label / Y Label**: 축 라벨 표시 토글
- **X Ticks / Y Ticks**: 축 눈금 표시 토글

**데이터 시리즈**:
- 체크박스로 개별 데이터셋 표시/숨기기
- 이름 입력창으로 범례 텍스트 변경 가능

**참조선**: LT95 ~ LT90 레벨을 점선으로 표시

**차트 색상**:
| 데이터 | 색상 |
|--------|------|
| REF LT-1 / REF IVL-1 | #4a9eff (파랑) |
| REF LT-2 / REF IVL-2 | #a855f7 (보라) |
| SAMPLE LT-1 / SAMPLE IVL-1 | #ef4444 (빨강) |
| SAMPLE LT-2 / SAMPLE IVL-2 | #f59e0b (주황) |

**COPY CHART**: 차트를 PNG 이미지로 클립보드에 복사 (미지원 시 다운로드)

### 4.6 IVL SPECTRUM 차트

- J ≈ 10 mA/cm² 지점의 **EL 스펙트럼** 표시
- X축: 파장 380~780 nm
- Y축: 강도 (a.u.)
- CSV의 14번째 컬럼(인덱스 13)부터가 파장별 스펙트럼 데이터
- 헤더 행의 해당 컬럼이 파장값(nm)

### 4.7 COPY (테이블 복사)

- 각 테이블 카드 우측 상단 **COPY** 버튼
- 클릭 시 테이블 내용을 **탭 구분 텍스트**로 클립보드에 복사
- Excel/Google Sheets에 바로 붙여넣기 가능

## 5. CSV 파일 형식

### 5.1 IVL CSV

```
헤더행: ... , [1]V, ... , [5]J(mA/cm²), ... , [7]EFF, ... , [9]EQE, [10]CIEx, [11]CIEy, ... , [13~]파장별_스펙트럼
데이터행: 측정값들 (쉼표 구분)
```

- 인덱스 5 (J값)이 10에 가장 가까운 행을 자동 선택
- 인덱스 13부터 스펙트럼 데이터 (헤더에 파장 nm 표기)

### 5.2 LT CSV

```
(메타데이터 행들)
헤더행: Hour, ... , [5]Luminance(%)
데이터행: 시간, ... , 휘도(%)
```

- 상위 6행 내에서 `hour` 키워드가 포함된 행을 헤더로 인식
- 인덱스 0 = 시간(hr), 인덱스 5 = 휘도(%)

## 6. 파일명 자동 매칭 규칙

파일명에서 다음 패턴을 순서대로 판별합니다:

1. **그룹 판별** (파일명 시작 부분)
   - `ref_` → REF
   - `sample_` 또는 `sam_` → SAMPLE
   - 그 외 → 매칭 실패

2. **타입 판별**
   - `ivl` + 숫자 → IVL 타입, 해당 번호
   - `lt` + 숫자 → LT 타입, 해당 번호
   - `ivl`만 있으면 → 파일명에 `2` 포함 시 2번, 아니면 1번
   - `lt` 또는 `수명` 키워드 → LT 타입

3. **예시**
   - `REF_IVL1_CH082_240224.csv` → REF IVL-1
   - `Sample_LT2_test.csv` → SAMPLE LT-2
   - `REF_LT2_CH082_T34_260224_G_Mono_PG267.csv` → REF LT-2

## 8. 기술 상세

- **Chart.js v4** (CDN): `https://cdn.jsdelivr.net/npm/chart.js@4`

### 8.2 주요 함수 목록

| 함수명 | 설명 |

| `init()` | 초기화: 그리드 렌더링, 이벤트 바인딩 |

## 9. 사용 흐름

```
1. 페이지 접속
2. CSV 파일 드래그 앤 드롭 (또는 클릭하여 선택)
3. FILE MATCHING 영역에서 자동 매칭 확인
   - 매칭 실패 파일은 📎 버튼으로 수동 지정
4. ANALYZE 버튼 클릭
5. 결과 확인:
   - IVL ANALYSIS 테이블에서 REF vs SAMPLE 비교
   - LT CHART에서 수명 그래프 확인 (슬라이더로 조절)
   - SPECTRUM 차트에서 EL 스펙트럼 확인
6. COPY 버튼으로 테이블/차트 복사 → Excel 등에 붙여넣기
```
